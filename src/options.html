Domain Filtering

How domain filtering works:
If the allowed list has entries, only those domains are captured. Everything else is ignored.
If the allowed list is empty, all domains are captured except those in the blocked list.
Subdomains match automatically — google.com matches docs.google.com.
If a domain appears in both lists, blocked wins.
